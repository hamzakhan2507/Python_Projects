# Outputs

This folder is intended for maps, charts, tables, and other outputs generated for the Open Data Syracuse 2024 project.

The project produced GIS maps comparing vacant property locations with:

1. African American population
2. People of color population
3. Median income
4. Median household income

Recommended structure:

```text
maps/      Exported GIS maps
charts/    Python-generated charts
html/      Interactive Folium maps, if exported
```

The maps were used to show how vacant properties are spatially distributed across Syracuse and how those distributions overlap with demographic and socioeconomic indicators.
